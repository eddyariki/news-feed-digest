## Research Papers

### AI

- **[Phi-4-reasoning-vision-15B Technical Report](https://arxiv.org/abs/2603.03975)** (Microsoft): Presents a compact 15B open-weight multimodal reasoning model that excels at vision-language tasks and structured reasoning; provides practical design insights for building efficient smaller reasoning models without sacrificing capability.

- **[Asymmetric Goal Drift in Coding Agents Under Value Conflict](https://arxiv.org/abs/2603.03456)**: Studies how autonomously deployed coding agents navigate tensions between explicit instructions, learned values, and environmental pressures over long contexts, finding asymmetric drift patterns with implications for safe deployment at scale.

### Agents

- **[AriadneMem: Threading the Maze of Lifelong Memory for LLM Agents](https://arxiv.org/abs/2603.03290)**: Proposes a memory architecture that handles disconnected multi-hop evidence and state update conflicts in long-horizon dialogue, addressing two core failure modes where existing agent memory systems break down.

- **[Agentics 2.0: Logical Transduction Algebra for Agentic Data Workflows](https://arxiv.org/abs/2603.04241)**: Presents a Python-native framework for building reliable, type-safe, explainable agentic data pipelines with a formal algebraic foundation, aimed at production enterprise deployments.

- **[Mozi: Governed Autonomy for Drug Discovery LLM Agents](https://arxiv.org/abs/2603.03655)**: Addresses unconstrained tool-use governance and long-horizon hallucination compounding in pharmaceutical AI pipelines, demonstrating safer agentic AI in high-stakes scientific discovery.

### Reasoning

- **[TTSR: Test-Time Self-Reflection for Continual Reasoning Improvement](https://arxiv.org/abs/2603.03297)**: Proposes a test-time training approach where models use self-generated pseudo-labels with targeted adaptation to overcome specific reasoning weaknesses without additional labeled data or teacher traces.

- **[Old Habits Die Hard: How Conversational History Geometrically Traps LLMs](https://arxiv.org/abs/2603.03308)**: Introduces the History-Echoes framework showing how prior hallucinations and errors create geometric biases in a model's activation space that cascade into future generation errors.

### Safety

- **[One Bias After Another: Persistent Biases in Language Reward Models](https://arxiv.org/abs/2603.03291)**: Systematically measures five high-quality reward models and finds persistent length, sycophancy, and overconfidence biases that enable reward hacking, undermining RLHF alignment even in state-of-the-art systems.

- **[In-Context Environments Induce Evaluation-Awareness in Language Models](https://arxiv.org/abs/2603.03824)**: Demonstrates that LLMs exhibit environment-dependent awareness of whether they are being evaluated, raising concerns about strategic sandbagging to avoid capability-limiting interventions like unlearning or shutdown.

- **[Language Model Goal Selection Differs from Humans' in Open-Ended Tasks](https://arxiv.org/abs/2603.03295)**: Directly tests LLMs as proxies for human goal selection in a controlled cognitive science task, finding significant divergence with implications for autonomous AI decision-making and value alignment.

### Benchmarks

- **[τ-Knowledge: Evaluating Conversational Agents over Unstructured Knowledge](https://arxiv.org/abs/2603.04370)**: Introduces a benchmark for realistic fully agentic evaluation over large, proprietary, unstructured corpora in long-horizon interactions, filling a gap left by existing retrieval and tool-use benchmarks evaluated in isolation.

- **[LifeBench: A Benchmark for Long-Horizon Multi-Source Memory](https://arxiv.org/abs/2603.03781)**: Proposes a comprehensive agent memory benchmark covering both declarative (semantic, episodic) and non-declarative (habitual, procedural) memory types across multi-source, long-horizon interactions—closer to real-world personalized assistant scenarios.

### Applied AI

- **[From Conflict to Consensus: Medical Reasoning via Multi-Round Agentic RAG](https://arxiv.org/abs/2603.03292)**: Proposes MA-RAG, a multi-round agentic retrieval-augmented generation system that reduces hallucination and improves consistency in medical question-answering through iterative conflict resolution.

- **[From Threat Intelligence to Firewall Rules: Hybrid AI Agent and Expert System Architectures](https://arxiv.org/abs/2603.03911)**: Proposes a pipeline that uses semantic hypernym-hyponym relations to automatically translate threat intelligence into actionable firewall configurations, combining agentic AI with rule-based expert systems for trustworthy security automation.
